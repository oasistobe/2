  • html要素

    • html要素はHTMLドキュメントのルートを表します。著者は、ルートhtml要素にlang属性を指定し、ドキュメントの言語を与えることをお勧めします。これにより、音声合成ツールを使用して使用する発音を決定したり、使用するルールを決定するための翻訳ツールなどを支援します。

    ソースコード 
    
        <!DOCTYPE html>
        <html>
        <head>
        <title>こんにゃく</title>
        </head>
        <body>
        <h1>こんにゃく</h1>
        <p>私はこんにゃくマスターである</p>
        </body>
        </html>
    実行結果 こんにゃく

    こんにゃく

    私はこんにゃくマスターである

  • head要素

    • head要素は、Documentのメタデータのコレクションを表します。著者 (author) の指定はいくつかの点で役立ちます。コンテンツに関する質問があって問い合わせしたい場合、誰がそのページを書いたのかに答えられます。 CMS (コンテンツ管理システム) によっては自動的にページの作者の情報を抽出して、このような目的で利用するための機能があります。

    ソースコード 
    
        <head>
        ・title要素
        ・meta要素
          ...など
        </head>
  • style要素

    • style要素を使用すると、作成者はドキュメントにCSSスタイルシートを埋め込むことができます。style要素は、スタイリング処理モデルへのいくつかの入力の1つです。この要素は、ユーザーのコンテンツを表すものではありません。

    ソースコード 
            <!DOCTYPE html>
          <head>
          <title>My favorite book</title>
          <style>
          body { color: black; background: white; }
          em { font-style: normal; color: red; }
          </style>
          </head>
          <body>
          <p>私の好きな食べ物は 
          <em>こんにゃく</em> 
          です。最近は
          <em>肉じゃが</em> 
          に
          <em>こんにゃく</em>
          を入れて食べました。
          </body>
          </html>
    ソースコード My favorite book

    私の好きな食べ物は こんにゃく です。最近は 肉じゃが に こんにゃくを入れて食べました。
